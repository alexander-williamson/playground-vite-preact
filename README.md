# Vite + Preact Demo

A compact demo of Preact (React but faster) + Vite (a static-site generator) framework.

## Localy development

```
npm install
npm run dev
```

## Build

```
npm install
npm run build
```

And if you want to serve it to test it locally:

```
npm install
npm run build
npx http-server ./dist
```
